# clota

An [IOTA](https://iota.org) Client Library Implemented in Clojure

## Usage

```bash
$ lein repl
> (clojiota.client/get-node-info)
> (clojiota.client/get-neighbors-activity)
> (clojiota.client/get-tips)
> ...
```

for more, look in `src/clota/client.clj`
